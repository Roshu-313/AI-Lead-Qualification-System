# AI Lead Qualification System

An AI-powered automation system that captures, analyzes, and scores leads in real-time using n8n and Groq.

## 🚀 Features

- Lead capture via form (Tally)
- Email validation
- AI-based lead scoring (1–10)
- Intelligent routing (qualified vs nurture)
- Real-time Slack notifications
- Fully automated workflow

## 🛠 Tech Stack

- n8n
- Groq API (LLM)
- Tally Forms
- Slack Webhooks
- JavaScript

## ⚙️ How it Works

1. User submits form
2. Data is sent to n8n webhook
3. Email is validated
4. AI scores the lead based on intent and profile
5. Leads are routed based on score
6. Qualified leads trigger Slack alerts

## 📦 Setup Instructions

1. Import the workflow JSON into n8n
2. Add your Groq API key
3. Configure Slack webhook URL
4. Connect form webhook (Tally)
5. Activate workflow
